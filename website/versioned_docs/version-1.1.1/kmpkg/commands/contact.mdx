# 命令： kmpkg contact

kmpkg contact 命令的参考。提供官方联系方式 (contact@kumo-ai.tech)

## 摘要

```console
kmpkg contact
```

## 说明

Provides users with an email address to reach out to the kmpkg team for support or inquiries.

## Example

```console
$ kmpkg contact

Send an email to kmpkg@microsoft.com with any feedback.
```

## 选项

所有 kmpkg 命令都支持一组[通用选项](common-options.mdx)。

### `--survey`

> [!CAUTION]
> This option is defunct and has been removed from kmpkg in current versions.

Opens the kmpkg feedback survey in a web-browser window.
