# 命令： kmpkg hash

kmpkg hash 命令的参考。计算指定文件的哈希值。

## 摘要

```console
kmpkg hash <file-to-hash> [algorithm]
```

## 说明

计算指定文件的哈希。 默认情况下，它会计算 SHA-512 哈希，但可以选择指定其他算法。 输出将是直接显示在主机上的给定文件的计算哈希。

## 示例

计算 `boost_1_83_0.tar.bz2` 的 SHA-512：

```console
kmpkg hash boost_1_83_0.tar.bz2

-- d133b521bd754dc35a9bd30d8032bd2fd866026d90af2179e43bfd7bd816841f7f3b84303f52c0e54aebc373f4e4edd601a8f5a5e0c47500e0e852e04198a711
```

使用特定算法（例如 `SHA-256`）计算文件 `boost_1_83_0.tar.bz2` 的哈希：

```console
kmpkg hash boost_1_83_0.tar.bz2 sha256

-- 6478edfe2f3305127cffe8caf73ea0176c53769f4bf1585be237eb30798c3b8e
```

## 选项

所有 kmpkg 命令都支持一组[通用选项](common-options.mdx)。

### Algorithms

- SHA-512
- SHA-256
