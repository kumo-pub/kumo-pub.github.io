# 函数： kmpkg_add_to_path

使用 kmpkg_add_to_path 将目录添加到 PATH。

将一个或多个目录添加到 PATH 环境变量。

## 使用情况

```cmake
kmpkg_add_to_path([PREPEND] [<path>...])
```

## 参数

### PREPEND

将项目加在路径前面而不是附加到后面。

### `<path>...`

要附加到 PATH 环境变量的路径。
