# 函数： kmpkg_minimum_required

断言用于生成端口的 kmpkg 程序的版本晚于提供的日期（包含当日）。

## 使用情况

```cmake
kmpkg_minimum_required(VERSION 2021-01-13)
```

## 参数

### VERSION

要对其检查的日期版本。
