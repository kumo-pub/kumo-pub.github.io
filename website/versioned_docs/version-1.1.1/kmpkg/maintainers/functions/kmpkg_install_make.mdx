# 函数： kmpkg_install_make

构建并安装 make 项目。

## 使用情况

```cmake
kmpkg_install_make(...)
```

## 参数

请参阅 [`kmpkg_build_make()`](kmpkg_build_make.mdx).

## 说明

此命令以透明方式转发到 [`kmpkg_build_make()`](kmpkg_build_make.mdx)，添加 `ENABLE_INSTALL`

- [x264](https://gitee.com/kumo-pub/kmpkg/blob/master/ports/x264/portfile.cmake)
- [tcl](https://gitee.com/kumo-pub/kmpkg/blob/master/ports/tcl/portfile.cmake)
