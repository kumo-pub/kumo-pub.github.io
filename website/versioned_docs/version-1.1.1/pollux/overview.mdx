# 概述

##  什么是pollux



Pollux 是由 Kumo 研发的高性能向量化执行引擎，旨在为数据密集型应用提供极致的查询处理速度。其核心设计基于现代硬件特性（如 SIMD 指令、缓存优化），
通过列式存储和向量化计算大幅提升分析型工作负载的效率。Pollux 支持跨平台部署，可作为独立库集成到 Presto、Spark 等计算框架中，
显著优化其执行性能。特性包括：

* ‌向量化处理‌：批量处理数据列，减少函数调用开销；
* ‌内存管理优化‌：减少碎片化并提升缓存命中率；
* ‌多线程并行‌：动态任务调度充分利用多核CPU；
* ‌扩展性‌：支持自定义函数和数据类型。
* Pollux 尤其适合实时分析、大规模 ETL, 搜索实时计算等场景，是 Kumo 内部及开源生态中提升数据处理效能的关键组件。
