# ktest 示例

如果您像我们一样，您想看看
[Kumo 示例](https://gitee.com/kumo-pub/ktest/blob/master/kumotest/samples)
示例目录中有许多注释良好的示例，展示了如何使用各种 ktest 功能。

* 示例 #1 显示了使用 googletest 测试 C++ 函数的基本步骤。
* 示例#2 显示了针对具有多个成员的类的更复杂的单元测试
功能。
* 示例 #3 使用测试夹具。
* 示例 #4 教您如何一起使用 kumotest 和 `kumotest.h`
充分利用这两个库。
* 示例 #5 将共享测试逻辑放入基本测试装置中，并在
衍生的固定装置。
* 示例 #6 演示了类型参数化测试。
* 示例#7 教授值参数化测试的基础知识。
* 示例 #8 显示在值参数化测试中使用`Combine()`。
* 示例 #9 显示使用侦听器 API 来修改 Kumo Test 的控制台
输出并使用其反射 API 来检查测试结果。
* 示例 #10 显示使用侦听器 API 来实现原始内存
泄漏检查器。
