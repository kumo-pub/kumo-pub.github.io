# 支持平台

KumoTest 需要符合 C++17 标准的代码库和编译器或
较新。

KumoTest 代码在以下平台上得到正式支持。下面未列出的操作系统或工具均受社区支持。为了
社区支持的平台，不会使代码复杂化的补丁可能是经过考虑的。

如果您发现平台有任何问题，请在[KumoTest Gitee 问题跟踪器](https://gitee.com/kumo-pub/ktest/issues)。
欢迎包含修复的 Pull 请求！

### 操作系统

*   Linux
*   macOS
*   Windows

### 编译器

*   gcc 5.0+
*   clang 5.0+
*   MSVC 2015+

**macOS 用户：** Xcode 9.3+ 提供 clang 5.0+。

### 构建系统

*   [CMake](https://cmake.org/)
