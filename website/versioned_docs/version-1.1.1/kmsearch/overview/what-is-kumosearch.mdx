# 什么是 `kumosearch`?

kumosearch 是一款 AI 驱动的端到端容错搜索引擎，专为即时搜索体验而设计，响应时间通常低于 50 毫秒。它显著提升了迭代效率，同时降低了开发难度和成本。

:::tip
kumosearch 与公共搜索引擎（例如 Google、Bing、Baidu等）存在显著差异。公共搜索引擎通过抓取网站数据供所有人进行搜索，而 kumosearch 的运作方式则有所不同。

kumosearch 不会主动抓取数据。相反，您可以将主数据库中的现有数据，或通过抓取工具获取的内容，以 CSV/JSON 文件的形式推送到 kumosearch。这样，您的用户便可通过您的网站或应用程序的界面，对这些数据进行搜索。
:::
