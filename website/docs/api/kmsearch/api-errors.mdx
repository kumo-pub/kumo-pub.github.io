# API 错误码

kumosearch API 使用标准 HTTP 响应码来指示请求的成功或失败状态。

2xx 范围的代码表示成功，4xx 范围的代码表示请求中的错误（例如，缺少必需的参数），5xx 范围的代码表示 kumosearch 服务本身出现错误。

| 状态码   | 描述                      |
|------|-------------------------|
| 400	 | 错误请求 - 由于语法错误，无法理解请求。   |
| 401	 | 未经授权 - 您的 API 密钥错误。     |
| 404	 | 未找到 - 请求的资源未找到。         |
| 409	 | 冲突 - 资源已存在。           |
| 422	 | 无法处理的实体 - 请求格式正确，但无法处理。 |
| 503	 | 服务不可用 - 我们暂时离线，请稍后再试。   |
