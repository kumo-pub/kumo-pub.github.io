# 函数： kmpkg_execute_in_download_mode

即使在下载模式下仍执行进程。

## 使用情况


```cmake
kmpkg_execute_in_download_mode(
    ...
)
```

此函数的签名与 `execute_process()` 相同。
如需了解详情，请访问 [`execute_process()`](https://cmake.org/cmake/help/latest/command/execute_process.html)。
