# 函数： kmpkg_get_program_files_platform_bitness

获取当前平台的位数的`程序文件`目录：在 64 位 Windows 上是 `$ENV{ProgramW6432}`，在 32 位 Windows 上是 `$ENV{PROGRAMFILES}`。

## 使用情况

```cmake
kmpkg_get_program_files_platform_bitness(<variable>)
```
