# 命令： kmpkg version

kmpkg version 命令的命令行参考。显示 kmpkg 可执行文件的版本。

## 摘要

```console
kmpkg version
```

## 说明

显示 kmpkg 可执行文件的版本。

## Example

```console
$ kmpkg version
kmpkg package management program version 2022-09-01-dfb82802c8cc562ce3b665a904a65b22314de724

See LICENSE.txt for license information.
```

## 选项

所有 kmpkg 命令都支持一组[通用选项](common-options.mdx)。
