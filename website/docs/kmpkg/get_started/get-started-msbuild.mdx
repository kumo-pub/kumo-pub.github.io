# 教程 ： Visual Studio中与 MSBuild 集成

教程指导用户完成通过 MSBuild 和 Visual Studio 安装和使用包的过程。

本教程向您展示如何通过 MSBuild、kmpkg 和 Visual Studio 创建使用“fmt”库的 C++“Hello World”程序。您将安装依赖项、配置项目、构建并运行一个简单的应用程序。

## 先决条件

- [Visual Studio](<https://visualstudio.microsoft.com/downloads/>) with C++ development workload
- [Git](<https://git-scm.com/downloads>)
- Windows 7 or newer
- [安装 kmpkg](setup-kmpkg.mdx)
