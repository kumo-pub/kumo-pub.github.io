# 为什么使用 kumosearch?

kumosearch 的设计目标是简化搜索应用的构建过程，实现高效迭代，显著缩短业务上线周期，并提供出色的即时搜索体验。

为实现这一目标，我们优化了系统架构和算法，确保每项功能都易于使用。通过简化搜索技术的部署和管理，开发人员能够轻松处理海量数据，
获得高速度、高精度的搜索结果。我们还为每个配置参数精心设计了合理的默认值，使得 kumosearch 在大多数用例中都能“开箱即用”，降低企业的开发成本和运维负担。

我们希望借助 kumosearch，使开发人员能够更自动化地构建和管理搜索应用，让您在开发过程中少担心一件事，并更快地实现卓越的搜索性能和智能的搜索体验。
